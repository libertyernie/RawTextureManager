Raw Texture Manager is designed to help you extract and replace the textures inside Super Smash Bros. Melee's uncompressed data files. It doesn't analyze the files or their structure in any way; rather, it tries to match the filename of the .dat file you're trying to open with one or more definition files in the Definitions folder. Each definition file lists the locations, formats, and sizes of each texture in the .dat file it is describing.

Unlike Melee Toolkit, this program does not find textures in a file on its own; it needs definition files. As such, it's essentially just a tool to automate the conversion and hex-editing process. In the future, Melee Toolkit (or a program like it) will likely make this program obsolete; until then, hopefully this will help fill some gaps (missing textures, etc.)

Definition file format

Definition files are in JSON, a data-interchange format derived from JavaScript and commonly used to transmit data on the web from one site to another.

Each definition file contains an object with three attributes:

  • Name: a filename or pattern to match. For example, PlMr??.dat will match PlMrNr.dat, PlMrBu.dat, and so on, but not PlMr.dat. When opening a file, Raw Texture Manager will use only definitions whose Name attribute matches the filename.
  • Description: shown in the Help → Available Definitions window.
  • Textures: An array of one or more textures in the file. Each texture definition has the following attributes:
    • Name: The name that the texture will appear under in Raw Texture Manager's list.
    • Type: The type of the texture. BrawlLib is used to extract and replace textures; common formats include CMPR and CI8.
    • Width: The width of the texture, in pixels.
    • Height: The height of the texture, in pixels.
    • Location: The offset in the file where the texture starts. Can be specified as a hexadecimal string, e.g. "0x14240".
    • Palette: An optional attribute (used for indexed formats like CI8 only.) This is an object with three additional attributes:
      • Type: The type of the palette; typically RGB565 or RGB5A3.
      • Colors: The number of colors in the palette.
      • Location: The offset in the file where the palettte starts. Can be specified as a hexadecimal string, e.g. "0x18240".

This version of Raw Texture Manager contains definitions for:

  • Roy (any color)
  • Kirby (any color)
    • Donkey Kong hat (any color)
    • Falco hat (any color)
    • Mewtwo hat (any color)
    • Jigglypuff hat (any color)
    • Mario hat (one file applies to all colors)
  • Link (any color)
  • Mario (any color)
  • Pikachu (any color)

When opening a custom texture, make sure the filename ends with the original Melee filename; i.e. MyTexture][PlMrNr.dat.
